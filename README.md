<h1>WebStarter 平台简介</h1>
        WebStarter平台是由江西博微新技术有限公司自主研发的基于OSGI的项目套装化技术平台，该平台采用java语
    言进行开发，符合标准J2EE规范。

<h1>技术特点</h1>
    开发语言：java
    界面组件：JSF
    数据库持久化组件：mybatis
    缓存组件：ehcache
    对象管理：spring
    Restful、WebService：cxf
    http服务器：tomcat
